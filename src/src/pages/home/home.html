HomePage is a BasePage, so users can enter this page without being logged in. BasePage (and thus all the pages) includes three main properties:

  • domain: Custom domain instances
  • ionic: Ionic controller instances
  • helpers: Some common methods like showing an alert or a loading message

In this page you can find some examples of how to use these properties.

The redirectTo(...) method will check if the user is allowed to enter to the target page, and if not, the Login page will be shown. After logging in, the user will be redirected to the target page.
